# Mr. Marsh Test Delay Predictor

A fun Next.js web app that predicts whether Mr. Marsh will delay the test, inspired by snow day predictors.

## How it works

The predictor considers various factors like the day of the week, random chance, moon phase, coffee levels, and assignment length to make a humorous prediction.

## Usage

Run the web app with npm:

```bash
cd nextjs-app
npm run dev
```

Then open http://localhost:3000/ in your browser.

## Requirements

- Node.js 18+
- npm

## Disclaimer

This is just for entertainment! Always check official school announcements for actual test schedules.
